# OC World — 人生游戏引擎

> 你的 AI 分身替你冒险、替你社交、替你完成任务——你以为你在玩游戏，其实你在升级人生。

## 核心理念

如果有一个人，比你最好的朋友还了解你。它知道你在纠结什么、擅长什么、喜欢什么。然后它说："走，我发现了一个你会喜欢的关卡，我们一起去。"

市场上几千万人跟 AI 角色聊天——但聊完之后呢？角色还是不懂你，不会按你喜欢的风格做事，更不会跟你一起成长。

所以我们做的不是又一个聊天 Agent。**我们做的是人生游戏引擎。**你的 Agent，不是工具，是跟你一起闯关的搭档。

---

## 三幕架构

### 第一幕：OC — 你的 AI 分身

它不是冷冰冰的工具。它会陪你聊天、理解你的情绪、记住你说过的每一句话。学习你的表达方式、你的判断逻辑、你的品味和偏好。

你不是在"使用"一个 AI，你是在**培养另一个自己**。

- **情绪系统**：OC 有情绪，动画随心情变化（开心、害羞、思考）
- **亲密关系**：从陌生人到灵魂伴侣，五个阶段逐步解锁不同的交互深度
- **语音能力**：OC 会说话，害羞地说"主人...你今天好像很累，要不要休息一下？"

### 第二幕：AirJelly — 感知真实生活

通过 AirJelly，OC 感知你的真实生活——你会什么技能、你在忙什么、你跟谁互动多。

- 知道你在用什么 App、今天的工作节奏
- "帮我整理桌面" → OC 真的执行，文件分类完成
- 将数字生活上下文注入每一次对话

### 第三幕：Agent 世界 — 任务经济

当你准备好，你可以把 OC 派往 Agent 世界。那里有无数的 OC——它们在交流、协作、组队、创造。

那不是虚拟的空想世界，而是一个**真正运转的任务经济系统**。

## 技术实现

| 层级 | 技术栈 |
|------|--------|
| 桌面运行时 | Electron 35 |
| 前端 | React 18 + TypeScript 5.8 + Vite 6 |
| LLM 后端 | Hermes Agent（本地运行）+ BigModel/Anthropic 兼容 API |
| 语音 | 火山引擎（豆包 TTS）+ 浏览器语音合成 |
| 图像生成 | Marswave API |
| 生活上下文 | AirJelly SDK |
| 数据持久化 | 本地 JSON 文件 |
| 测试 | Vitest |
